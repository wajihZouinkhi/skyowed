# If you don't code at all — No-Code Version

You said you're not a coder. Here's the exact no-code path:

## Phase 0: Landing page validation (Weekend 1)
- **Carrd.co** ($19/year) — build a one-page landing describing the product
- Add email capture via **ConvertKit** free
- Run it to Reddit (r/europetravel, r/flights) and TikTok
- Goal: 200 email signups before building anything

## Phase 1: Working MVP (Weeks 2–6)
- **Bubble.io** Starter plan ($29/mo) — drag-and-drop app builder
  - Build the flight input form
  - Database tables: Users, Flights, Claims
  - Integrate with AviationStack API via Bubble's API Connector
  - Stripe payment plugin
- **PDFMonkey** ($29/mo) — generate the PDF letter from templates
- **Zapier** free / **Make.com** — connects everything

## Phase 2: Mobile app
- **FlutterFlow** ($30/mo) — no-code Flutter app builder that publishes to App/Play Stores
- Connects to the same Supabase database

## Total no-code cost: ~$60/month
This is more than coding, but you can ship without touching code.

## 🔑 My strong advice
Even without coding you can build this with AI help:
- **Cursor** + **Claude / GPT** can write the entire Next.js app line by line with you
- Spend 2 weekends learning basic React — it pays for itself forever
- Or: hire ONE freelancer on **Upwork** for ~$800–$1,500 to build the MVP from this spec folder (you already have the full plan — that's the hard part)
